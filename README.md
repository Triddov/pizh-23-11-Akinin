## 🚀 How to Run it

1. Clone the repository  
   ```bash
   git clone https://github.com/Triddov/pizh-23-11-Akinin
   cd pizh-23-11-Akinin/laba{number}
   ```
2. And just run it!
	```bash
	python {task_name}.py
	```

Replace {number} with the required lab number and {task_name} with the assignment number
